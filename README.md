# Responsive Semantic Web

A responsive and semantic HTML/CSS website project showcasing a clean, accessible layout.  
This project demonstrates proper HTML semantics, responsive design, Bootstrap integration, and basic JavaScript functionality for a contact form and hero interactions.

## Features

- Fully responsive design using **Bootstrap 5**
- Semantic HTML elements for accessibility and SEO
- Hero section with a **call-to-action button** and modal
- Contact form with **validation and success/error messages**
- Smooth scrolling for navigation links
- Scroll animations for cards, tables, and quotes
- Testimonial rotator for mobile view
- Dynamic footer year using JavaScript

## Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript (Vanilla JS)
